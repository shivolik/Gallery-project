# Gallery Project

A modern and responsive image gallery built with **React**, **Vite**, and **Tailwind CSS**. The project provides a clean interface for displaying images with a responsive layout and smooth user experience.

## 🚀 Features

* Responsive gallery layout
* Built with React components
* Styled using Tailwind CSS
* Fast development with Vite
* Clean and modern UI
* Easy to customize

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/shivolik/Gallery-project.git
```

Navigate to the project folder:

```bash
cd Gallery-project
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local server URL shown in the terminal (usually `http://localhost:5173`).
